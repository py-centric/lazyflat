Usage
=====

Run the application:

.. code-block:: bash

   ./target/release/lazyflat

Keybindings
-----------

.. list-table::
   :widths: 20 80
   :header-rows: 1

   * - Key
     - Action
   * - `q` / `Esc`
     - Quit / Close Help
   * - `?`
     - Toggle Help
   * - `/`
     - Enter Search Mode
   * - `r`
     - Refresh Data
   * - `h` / `Left`
     - Previous Tab
   * - `l` / `Right`
     - Next Tab
   * - `j` / `Down`
     - Next Item
   * - `k` / `Up`
     - Previous Item
   * - `x`
     - Uninstall Selected
   * - `u`
     - Update Selected
   * - `U`
     - Update All
   * - `i`
     - Install Selected (Discover Tab)
   * - `Enter`
     - Confirm Search (Discover Tab)
