Installation
============

Prerequisites
-------------

- `Rust <https://www.rust-lang.org/tools/install>`_ (latest stable version)
- `flatpak` CLI installed on your system.

Build from Source
-----------------

.. code-block:: bash

   git clone https://github.com/yourusername/lazyflat.git
   cd lazyflat
   cargo build --release

The binary will be available at `target/release/lazyflat`.
